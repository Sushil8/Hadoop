Steps to run this project on Linux(Ubuntu 16.04)

1. Clean the dataset using R
script to run - pre.r

2. Install Hadoop, Pig, Hive and Spark

3. Install database systems MySql and Cassandra

4. To do fifa 18 players analysis in different aspects, run pig, hive and Spark queries. The output of the respective will be stored in the database system or locally.
